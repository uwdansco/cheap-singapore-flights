# Cheap Singapore Flights

Flight price tracking and alert platform monitoring prices from Singapore (SIN) to 50+ destinations worldwide.

## Features

- Personalized price tracking with custom thresholds
- Email alerts when prices drop
- Mobile-first PWA with offline support
- Comprehensive admin dashboard
- Price analytics and history

## Tech Stack

- React 18 + TypeScript + Vite
- TailwindCSS + Shadcn/ui
- Lovable Cloud (Supabase)
- Amadeus Flight API
- Resend Email

## Getting Started

```bash
npm install
npm run dev
```
